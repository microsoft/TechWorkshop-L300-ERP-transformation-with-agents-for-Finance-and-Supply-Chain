# Exercise 02: Configure shared resources

For each of the agents, you need to configure an identity (user account). The Expense agent and Supplier Communications agent each need a mailbox to operate correctly.

## Objectives
After you complete this exercise, you'll be able to:

-   Create identities (user accounts) for agents
-   Create shared mailboxes for agents

**Estimated time**: 20 minutes

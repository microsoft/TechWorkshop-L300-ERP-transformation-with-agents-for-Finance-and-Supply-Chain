# Exercise 03: Configure the Supplier Communications agent

You've completed all the preliminary configuration steps. Now it's time to configure the agents themselves.

In this exercise, you'll configure the Supplier Communications agent.

## Objectives
After you complete this exercise, you'll be able to:

- Assign roles to the agent identity
- Enable required features
- Configure the shared mailbox
- Create and configure connections
- Link an email address to a vendor
- Configure and activate the agent
- Validate functionality
- Troubleshoot issues

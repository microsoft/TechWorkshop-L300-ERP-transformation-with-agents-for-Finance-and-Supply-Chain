# Task 02: Enable required features
<!-- Estimated duration 7 minutes-->

## Introduction
The Supplier Communications agent integrates with a variety of other resources and services. You need to ensure that features required by the agent are enabled.

## Description
In this task, you'll enable the features that are required to support the Supplier Communications agent.

## Success criteria
The following features are enabled:

- (Production ready preview) Immersive Home
- (Production Ready Preview) Agent management
- (Production Ready Preview) Account reconciliation agent


## Key tasks
### 01: Enable required features

1. In **Finance and Operations**, go to the **Feature management workspace**.

    ![gykzgghi.jpg](../../media/gykzgghi.jpg)

1. On the menu bar, select **Not enabled**.

1. In the **Filter** field, enter **(Production ready preview) Immersive Home**. 

    {: .note }
    > It's important that you search for the entire name (including parentheses) or the tool may not find the feature.

    ![mfohgtmu.jpg](../../media/mfohgtmu.jpg)

1. In the **(Production ready preview) Immersive Home** pane, if the feature is not already enabled, select **Enable now**.

    {: .note }
    > If you completed Exercise 03 to configure the Supplier Communications agent, the Immersive Home feature should already be enabled.

    ![j9p1y15q.jpg](../../media/j9p1y15q.jpg)

1. In the **Enable feature** pane, select **Enable**.

    ![sjdavgiq.jpg](../../media/sjdavgiq.jpg)
    ![mlcuqfd7.jpg](../../media/mlcuqfd7.jpg)

1. Repeat steps **3** and **4** to add the following features. 

    {: .warning }
    > It's important that you enable each feature in the order presented.
    >
    > As you attempt to enable new features, you may see a message that resembles the following screenshot. The message indicates the feature which you previously enables is not quite ready yet.
    >
    > ![gzk5smag.jpg](../../media/gzk5smag.jpg)
    >
    > If you completed Exercise 03 to configure the Supplier Communications agent, the Immersive Home and Agent Management features should already be enabled.
    >
    > - **(Production Ready Preview) Agent management** 
    > - **(Production Ready Preview) Account Reconciliation Agent**

# Exercise 00: Request and validate a Finance and Operations Demo environment

This exercise serves as a reference for the process that you'll need to follow to request a Finance and Operations Demo environment in your demo environment.


## Objectives
After you complete this exercise, you'll be able to:

- Access the [Biz Apps Demo Hub website](https://bizappsdemos.microsoft.com/newdemorequest?category=accelerate+revenue+generation "Biz Apps Demo Hub website").
- Request a **Finance and Operations Demo** environment 

## Duration

**Estimated time**: 10 minutes

## Tasks
This exercise includes the following tasks:

- Request a Finance and Operations Demo environment
- Confirm that the environment meets the minimum version requirement to support the agents
